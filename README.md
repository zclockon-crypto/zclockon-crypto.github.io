# zclockon-crypto's AI Portfolio

欢迎来到我的个人 AI 项目展示网站！

## 🌟 网站特性

- ✨ **现代化设计** - 响应式布局，完美适配各种设备
- 🎨 **流畅动画** - 平滑的过渡和交互效果
- 📱 **移动友好** - 在手机、平板和桌面上都能完美显示
- 🚀 **快速加载** - 优化的代码和资源
- 🎯 **易于自定义** - 简单修改项目信息和内容

## 📖 如何自定义

### 1. 编辑项目信息
编辑 `index.html` 文件中的项目卡片：
- 修改项目名称、描述和链接
- 添加或删除项目卡片
- 自定义标签和技术栈

### 2. 更新个人信息
- 修改"关于我"部分的个人简介
- 更新联系链接（GitHub、邮件、Twitter、LinkedIn）
- 调整统计数据

### 3. 自定义样式
编辑 `styles.css` 文件：
- 修改颜色主题（`--primary-color`, `--secondary-color` 等）
- 调整字体大小和间距
- 更改布局和响应式断点

## 🎨 颜色主题

当前使用的配色方案：
- 主色：`#667eea`
- 副色：`#764ba2`
- 强调色：`#f5576c`

你可以在 `styles.css` 中的 `:root` 选择器中修改这些颜色。

## 📝 项目模板

每个项目卡片包含：
- 项目名称
- 项目描述
- 技术标签
- "查看代码"链接（指向 GitHub）
- "演示"链接（指向在线演示或项目页面）

## 🚀 部署

这个网站已经部署在 GitHub Pages 上。

**域名**: `https://zclockon-crypto.github.io`

任何推送到 `main` 分支的更改将自动部署。

## 📚 更新步骤

1. 克隆或编辑仓库
   ```bash
   git clone https://github.com/zclockon-crypto/zclockon-crypto.github.io.git
   cd zclockon-crypto.github.io
   ```

2. 修改 `index.html`、`styles.css` 或 `script.js`

3. 提交更改
   ```bash
   git add .
   git commit -m "Update portfolio"
   git push origin main
   ```

4. 访问网站查看更新（可能需要等待几秒钟）
